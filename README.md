# Layout Indicator — индикатор раскладки у курсора

Маленький полупрозрачный индикатор следует за курсором: **нейтральная точка**
+ **цветное кольцо** и ripple-волны — раскладку видно всегда.

> **Адаптация цвета к фону временно отключена** (`ADAPTIVE_BG = False`) — см. Issues на GitHub.

| Раскладка | Ripple |
|-----------|--------|
| EN | indigo |
| RU | rose |
| DE | amber |
| FR | sky |
| UK | violet |
| ES | coral |

## Скачать готовый exe

Папка **`Releases/`** — портативный `LayoutIndicator.exe`, без установки.

## Установка на любой ПК (рекомендуется)

Готовый установщик: **`installer\LayoutIndicator-Setup-1.0.exe`**

1. Скопируйте этот файл на нужный ПК и запустите.
2. Установка идёт для текущего пользователя — **права администратора не нужны**.
3. В мастере есть галочки:
   - **Запускать при входе в Windows** (автозапуск) — включена по умолчанию;
   - создать ярлык на рабочем столе;
   - запустить сразу после установки.

Python на целевом ПК **не требуется** — exe самодостаточный.

Удаление — через «Параметры → Приложения» или ярлык «Удалить Layout Indicator».

## Портативный запуск (без установки)

Можно просто скопировать `dist\LayoutIndicator.exe` и запустить двойным кликом.

## Автозапуск (управление вручную)

Автозапуск реализован через ветку реестра `HKCU\...\Run`. Управлять можно так:

```powershell
# включить
.\dist\LayoutIndicator.exe --enable-autostart
# выключить
.\dist\LayoutIndicator.exe --disable-autostart
# проверить статус (из исходника, где есть консоль)
python layout_indicator.py --autostart-status
```

Установщик при выборе галочки сам добавляет запись автозапуска и удаляет её при
деинсталляции.

## Остановка

- Сочетание клавиш **Ctrl + Alt + Q**.
- Либо `stop.bat` (для запуска из исходника), либо завершить `LayoutIndicator.exe`
  в Диспетчере задач.

## Запуск из исходника

```powershell
pythonw layout_indicator.py     # без окна консоли
# или
python layout_indicator.py      # с консолью (видно сообщения CLI)
```

Используется только стандартная библиотека (`tkinter`, `ctypes`, `winreg`).

## Особенности

- Точка «сквозная»: клики мыши проходят сквозь неё и не мешают работе.
- Не отображается в панели задач и в Alt+Tab.
- Раскладка берётся из системной локали активного окна — работает с любой
  раскладкой Windows.

## Настройка внешнего вида

Параметры в начале `layout_indicator.py`:

| Параметр     | Назначение                                  |
|--------------|---------------------------------------------|
| `DOT_SIZE`   | диаметр центральной точки (px)               |
| `CANVAS_SIZE`| размер холста под ripple-волны (px)          |
| `OFFSET_X/Y` | смещение относительно курсора (px)           |
| `OPACITY`    | прозрачность (0..1), сейчас 0.55             |
| `ADAPTIVE_BG` | адаптация к фону (сейчас `False`, см. issue)  |
| `BG_LUMA_FIXED` | яркость фона для палитры, если адаптация выкл. |
| `HUES`       | оттенок (hue) для каждой раскладки, 0..1     |
| `TICK_MS`    | интервал курсора и анимации (мс)             |
| `LAYOUT_POLL_MS` | интервал опроса раскладки (мс)           |
| `FOLLOW_LERP` | плавность следования (0..1), без отскока      |
| `FOLLOW_SNAP` | зона покоя — гасит дрожание на месте (px)    |
| `RIPPLE_MS`  | длительность волны (мс), сейчас 580          |
| `RIPPLE_FADE_POWER` | скорость затухания волны (>1 быстрее)  |
