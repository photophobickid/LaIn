# Releases

Портативные сборки Layout Indicator (LaIn).

| Файл | Описание |
|------|----------|
| `LayoutIndicator.exe` | Самодостаточный exe, Python не нужен |

Запуск: двойной клик. Выход: **Ctrl + Alt + Q**.

Текущая версия — ripple dot, монохромная точка 14 px + цветное кольцо раскладки, плавный follow.
